# SistemaAcademico

Projeto desenvolvido em Java com interface gráfica Swing e banco de dados MySQL.

## Funcionalidades

- Cadastro de alunos
- Consulta de alunos por RGM
- Alteração de dados do aluno
- Exclusão de aluno
- Cadastro de notas e faltas
- Boletim acadêmico
- Validação de campos
- Integração com banco de dados MySQL
- Exclusão em cascata de notas e faltas ao remover aluno

## Tecnologias utilizadas

- Java
- Java Swing
- MySQL
- JDBC
- Eclipse IDE
